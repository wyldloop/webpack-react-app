# Webpack React Starter

## Initial Setup

```
$ git clone git@github.com:rcaweb/webpack-react-app.git
$ cd webpack-react-app

# need to create the build folder
$ mkdir dist

# installs all dependencies
$ yarn install

# this will bundle everythying
$ npx webpack 

# builds dev version of app and will launch in browser http://localhost:9000/
$ yarn run dev

# builds prod version for deployment
$ yarn run prod

# will build prod version and also deploy to gh-pages branch
$ yarn run deploy
```
